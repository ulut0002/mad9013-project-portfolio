Some of my projects

  • Muzika

    Muzika is a music streaming app that allows users to listen to their favorite tracks on demand. Designed with Figma, the app offers a sleek and intuitive user interface that makes it easy to find and play music

  • JamStack

    JamStack is a single page web application that is built using JavaScript and connects to an external API to retrieve movie details. The application is a client-side app, meaning that it runs entirely in the user's web browser and does not require a server to operate

  • Cat Overload

    The single-page app utilizes a grid box layout and connects to a cat image API to provide users with a visually appealing and engaging experience. The animations and user-friendly interface make it easy for anyone to access and enjoy the endless supply of cat pictures available through the API
